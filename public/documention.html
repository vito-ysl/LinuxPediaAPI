Documentação da Linux Pedia API

Bem-vindo à documentação da Linux Pedia API! Aqui você encontra exemplos de requisições, respostas e instruções básicas para começar a usar a API.

Base URL

"https://linux-pedia-api.vercel.app/api/v1/comandos/${categoria}"

Endpoints por Categoria

/api/v1/comandos/arquivos

Método: GET

Descrição: Lista comandos relacionados a arquivos.

[
  { 
    "comandos": "ls",
    "descricao": "Lista arquivos e diretórios",
    "exemplo": "ls -la" 
  },
  { 
    "comandos": "cp",
    "descricao": "Copia arquivos ou diretórios",
    "exemplo": "cp arquivo.txt /tmp"
  }
]

/api/v1/comandos/sistemas

Método: GET

Descrição: Lista comandos do sistema.

[
  { 
    "comandos": "uname", 
    "descricao": "Exibe informações do sistema", 
    "exemplo": "uname -a" 
  },
  { 
    "comandos": "top", 
    "descricao": "Mostra processos em execução", 
    "exemplo": "top" 
  }
]

/api/v1/comandos/rede

Método: GET

Descrição: Lista comandos de rede.

[
  { 
    "comandos": "ping", 
    "descricao": "Testa conectividade de rede", 
    "exemplo": "ping 8.8.8.8" 
  },
  { 
    "comandos": "ifconfig", 
    "descricao": "Configura interfaces de rede", 
    "exemplo": "ifconfig eth0" 
  }
]

/api/v1/comandos/pacotes

Método: GET

Descrição: Lista comandos de gerenciamento de pacotes.

[
  { 
    "comandos": "apt-get", 
    "descricao": "Gerencia pacotes no Debian/Ubuntu", 
    "exemplo": "apt-get update" 
  },
  { 
    "comandos": "yum", 
    "descricao": "Gerencia pacotes no CentOS/Fedora", 
    "exemplo": "yum install nano" 
  }
]

/api/v1/comandos/util

Método: GET

Descrição: Lista comandos utilitários.

[
  { 
    "comandos": "echo", 
    "descricao": "Exibe mensagens no terminal", 
    "exemplo": "echo Olá Mundo" 
  },
  { 
    "comandos": "date", 
    "descricao": "Mostra a data e hora atuais", 
    "exemplo": "date" 
  }
]

Todos os endpoints retornam dados em formato JSON.
