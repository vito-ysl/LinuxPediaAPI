Termos de Uso da Linux Pedia API

Ao utilizar a Linux Pedia API, você concorda com os seguintes termos:

  • Uso Livre: A API é gratuita e aberta para uso pessoal, educacional e comercial, desde que respeitados estes termos.
  • Limite de Requisições: Para garantir a estabilidade do serviço, evite realizar requisições excessivas ou automatizadas sem necessidade.
  • Propriedade Intelectual: O conteúdo fornecido pela API é de uso livre, mas recomenda-se citar a fonte ao utilizar em projetos públicos.
  • Responsabilidade: A equipe da Linux Pedia API não se responsabiliza por eventuais danos, perdas ou prejuízos decorrentes do uso da API.
  • Alterações: Os termos de uso podem ser alterados a qualquer momento sem aviso prévio. Consulte esta página periodicamente.
  • Contribuições: Contribuições são bem-vindas via pull request no GitHub.
  • Contato: Para dúvidas ou sugestões, entre em contato pelo nosso repositório ou pelo e-mail contato@linuxpediaapi.com.
